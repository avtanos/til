Как вводить данные для `окуу()`: каждая операция читает одну строку; для чисел/логики берётся 1-й токен, для `сап` — вся строка, для `тизме<тип>` — значения через пробелы. `окуу()` үчүн маалыматты киргизүү: ар бир чакыруу бир сапты окуйт; сандар/логика үчүн 1-токен, `сап` үчүн бүт сап, `тизме<тип>` үчүн пробел менен бөлүнгөн маанилер.
